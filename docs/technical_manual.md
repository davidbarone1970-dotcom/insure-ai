# Technical Manual

## Setup
1. Open the folder in VS Code.
2. Create a virtual environment.
3. Install dependencies.
4. Run the simulation.

## Entry points
- `simulation/run_simulation.py` for local simulation
- `api/app.py` for future API usage

## Extensibility
- Replace rule-based logic in `agents/` and `models/`
- Integrate external APIs through `api/`
- Add workflow exports in `workflows/`
