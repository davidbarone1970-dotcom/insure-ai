# Solution Overview

## Target architecture
- Channel layer: web, broker, call center
- Operational layer: CRM, claims, policy management
- Integration layer: API and workflow orchestration
- Intelligence layer: agents, rule engine, optional LLM/ML services
- Insight layer: BI and KPI monitoring

## Agent roles
- LeadAgent: evaluates commercial potential
- ClaimsAgent: classifies and routes claims
- RetentionAgent: flags churn risk and proposes action
- Orchestrator: consolidates results and decides next best action
